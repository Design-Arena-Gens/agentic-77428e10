## Docteur Robot — Clinique de Demain

Application web immersive en Next.js pensée pour préparer une visite chez un docteur robot. Elle propose un parcours narratif et interactif autour de la médecine augmentée :

- **Héros futuriste** : présentation du docteur robot, métriques en direct et storytelling.
- **Tableau biométrique** : indicateurs clés (cohérence cardiaque, alignement neural, confort patient...).
- **Modules immersifs** : cartes détaillant les expériences sensorielles et diagnostiques proposées.
- **Timeline protocolaire** : aperçu étape par étape de la consultation augmentée.
- **Planificateur intelligent** : sélection du motif de visite, choix de la date et du créneau, briefing automatisé.
- **Retours d'expérience & FAQ** : témoignages patients et réponses aux questions fréquentes.

## Démarrage rapide

```bash
npm install
npm run dev
```

Ouvrez ensuite [http://localhost:3000](http://localhost:3000) pour découvrir l'interface. Les modifications sont rechargées automatiquement.

## Scripts disponibles

- `npm run dev` — lance le serveur de développement.
- `npm run build` — génère la version de production.
- `npm run start` — démarre un serveur avec le build produit.
- `npm run lint` — vérifie la qualité du code.

## Déploiement

Le projet est optimisé pour un déploiement instantané sur Vercel. Compilez avec `npm run build`, puis déployez via `vercel deploy` selon votre configuration.
